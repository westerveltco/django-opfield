## Requirements

- Python 3.8, 3.9, 3.10, 3.11, 3.12
- Django 4.2, 5.0

## Getting Started

1. Install the package from PyPI:

```bash
python -m pip install django-opfield
```

2. Add the app to your Django project's `INSTALLED_APPS`:

```python
INSTALLED_APPS = [
    ...,
    "django_opfield",
    ...,
]
```
